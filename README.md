# devsecops-mini-project
Mini DevSecOps project to demonstrate basic security checks in CI/CD.

## DevSecOps Security Pipeline

- Implemented GitHub Actions CI pipeline
- Integrated TruffleHog for secret scanning
- Pipeline automatically scans on every push
- Configured fail gate for verified secrets
- Tested pipeline using dummy AWS key
- Observed tool verification behavior
- Removed secret as part of remediation
